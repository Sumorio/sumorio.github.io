自建VPS科学上网节点（新手)

什么是域名？如何自行搭建 VPS？SSL 申请？..

一、前期准备

1）注册购买域名

什么是域名？

  • 域名可以说是网址的小名，目的是为了便于记忆。
  • 例如，小明.com 是一个域名。大家可以直接访问 小明.com 来代替 IP 地址，然后域名系统（DNS）会将 小明.com 转化成便于机器识别的 IP 地址。这样，大家只需记忆 小明.com 这一串带有好记忆的字符即可访问网址，而不需要记忆没有含义的数字。

域名购买

推荐使用 namesilo 网站注册，因为该网站的 whois 服务是免费的.
什么是whois 服务，简单来说，类似匿名购买，这项功能使得其他人无法查询是你购买了该域名，有些网站该功能为增值服务需要单独付费。

2）购买 VPS

VPS 是什么？

  • VPS(Virtual Private Server 虚拟专用服务器)技术，将一台服务器分割成多个虚拟专享服务器的优质服务。每个 VPS 都可分配独立公网 IP 地址、独立操作系统、独立超大空间、独立内存、独立 CPU 资源、独立执行程序和独立系统配置等。
  • 例如，VPS 把 1 个大房子分割成 2 个小房间，国内的房间 A 和国外的房间 B，每个小房间都有自己的地址&系统&资源。现在我想访问 youtube，但是我所在的房间 A 里没有 YouTube，这时候就需要我先进入国外的房间 B，才能访问房间 B 里才有的 youtube。此时，国外的房间 B 就相当于一个 VPS 中转站。

3）注册 CloudFlare

什么是CloudFlare?

是一个内容分发网络，也就是一个 CDN，对于搭建节点来说，就是一个转发的服务器。

常规来说，我们需要本地服务器和网络服务器的数据进行交换，但是有了 CloudFlare 之后，数据可以通过 CloudFlare 进行数据中转，优秀的 VPS 代理线路通常不需要 CloudFlare 进行数据转发，但是如果购买不好的 VPS，通常需要使用 CloudFlare 进行数据过滤和转发，能大大提升访问速度。

账号注册

官网：https://dash.cloudflare.com/sign-up

域名托管

使用 CloudFlare 托管域名，接管域名管理的业务。我们也可以不托管域名，但是域名托管可以让网站的 DNS 域名解析的生效时间大大缩短。

  1. 将第一步购买的域名，新增到域名列表中，选择免费的服务即可。
  2. 更改域名的名称服务器；
  3. 等待域名托管生效（5 分钟左右）
  4. 成功后，今后所有关于域名的设置，均可使用 cloudflare 实现。

绑定域名和公网IP

  1. 设置 DNS ➡️ 添加记录 ➡️ 生成二级域名（建议使用二级域名）；
  2. 检测域名是否成功：win+r➡️cmd➡️ping 二级域名；
  3. 如果 ping 不通，有 2 个原因，vps 没上线 or 域名被防火墙封了（联系 vps 客服更换 ip）.
